迭代器

迭代器（Iterator）区别于可迭代对象（Iterable）：后者只要能用于for循环就是，前者可以用next()并产生对应的值。

list，dict，str都是Iterable对象，但不是Iterator对象。
而generator是Iterator对象，当然也是Iterable对象。
